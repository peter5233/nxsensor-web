HOHWave™ Wireless Mesh Network


Overview

The premise of achieving smart solar power solution is data communication. The NxSensor system adopts a wireless mesh network - HOHWave™. It is explicitly designed for wireless sensor network of high density and large area, just like the case of PV module monitoring in solar power system. It offers advantages like self-organization, multi-hop, flexibility, ease of use and reliability, making it capable for extensive PV assets measurements and event alerts.

Comparison

    The conventional way of PV module-level data communication is ether by PLC or ZigBee. The main drawbacks are:
  • For PLC, it needs to deploy a data reader on each string to receive data sent from each individual PV module, thus increasing the overall solution costs.
  • For ZigBee, the number of sensors that a network can access or accommodate is usually less than 300 nodes, so applications based on ZigBee is limited by its difficulty to expand the network capacity.

How it works

After the HOHWave™ network has started to work, a sensor broadcasts its status to the surroundings and receives messages from others. After a while, all the sensors in the network could detect each other and know the neighbors. Next, the network forms into a hierarchical structure, for example, the gateway has the smallest level, aka, level 0; the sensors within the gateway's signal coverage are the level 1; the sensors in the outside rings but within level 1's signal coverage are the level 2. Eventually, every sensor has its level index, and data transmission could be made by flooding or routing strategies. Compared to Zigbee, the data transmission path between one sensor and the gateway is not singular or fixed, thus making the network more robust and require less manual configuration.
