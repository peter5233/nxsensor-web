PV Module-Level Monitoring


Overview

Module-level monitoring is an innovative approach for owners and O&M companies to improve financial performance and reduce risk in managing PV plants. The accuracy and insight from this new solution, with the aids of intelligent cloud based data analysis and diagnostics, bring more opportunities to improve solar energy yields and improve O&M efficiency.

Why Module-level Monitoring?

PV module is the source of energy production. However, many factors could affect its performance, for example, trees growing around and casting shadows on the PV modules, or birds throwing rocks on the surface and causing broken PV modules, or simply because of the module's self degradation over lifetime - all these factors could deteriorate the performance of a PV module.

Since all the PV modules in a string are connected in series, if one module fails, the power generation efficiency of an entire string will be reduced. It is estimated that 0.6% of PV modules stop working properly in a year, the compounding effects of the faults would actually lose more in a plant basis. If faults aren't be solved immediately, they would continue causing losses in the remaining time. Therefore, for O&M activities, the challenge is how to find these faults precisely and quickly among the large-scale PV modules.

PV O&M challenge: how to find faults precisely and quickly?

The NxSensor module-level monitoring system measures the performance of each individual PV module with great precision. The resulting data enables intelligent cloud software to identify module faults, diagnose root causes and recommend O&M activities. By comparison, inverters only monitor the solar power system at string level. Due to the low monitoring accuracy, it is impossible for inverters to help O&M team confirm the faults or suggest a maintenance action.

    With the NxSensor monitoring system, it allows:
  • Module performance measurement: tracking module-level performance over lifetime, recognizing potential module faults;
  • Performance risk management: the increased assurances of a solar power system that delivers expected power production;
  • Better reliability / safety: identifying erroneous electrical connection faults, preventing problems before they taking system down;
  • Efficient O&M workflow: increasing the efficiency of O&M activities by assessing performance precisely and digitizing PV assets;

System Components

The NxSensor system consists of four components:

The NxSensor system components

(1) The sensor unit

The sensor unit is integrated into the junction box of a PV module, it continuously measures the voltage of the module output and the temperature of the junction box, offering insight into module-level performance.

(2) The gateway

The sensor unit transmits the data to the gateway in a wireless way. The gateway collects and transmits the data to the cloud platform. The gateway and the sensor units form a mesh network - HOHWave™, which is a wireless technology enabling low-cost and extensible network for module-level data communication.

(3) The cloud platform

The cloud platform analyzes the data in machine learning algorithms, based on which to indicate the performance variations among PV modules and find the potential faults. It also provides the data interfaces for 3rd-party O&M companies to integrate data or further analyze data.

(4) The user terminals

Users could access the data from web-based devices like mobile phones and PCs.

Features

  • Measures the performance of each module individually;
  • Uses wireless mesh technology to transmit module data;
  • Enables cloud-based AI machine learning data analysis;
  • Alerts when fault happens or power production declines;
  • Provides fault location and diagnostic info for O&M teams;
  • Stand-alone monitoring solution with lower overall price;

Benefits

Yield more and save more

For installations, modules can be setup incorrectly and plugs forgotten, the NxSensor monitoring system provides extensive and detailed analysis of a PV plant's installation, avoiding off-line tests to troubleshoot problems or to check module consensus issues, enforcing a better quality of PV plant installations.

For plant owners and operators, defective modules or unsolved issues may have been existing for years, the NxSensor monitoring system detects hidden faults that lead to power losses or low efficiency, enables the taking of immediate O&M actions, thus avoiding revenue losses and increasing yields.

For maintenance, O&M teams are able to find failed modules with great certainty. They can determine the appropriate remedy before sending out a crew. It is even possible to prevent major problems before they take system down. Such integration will lead to the development of smart O&M strategies.
